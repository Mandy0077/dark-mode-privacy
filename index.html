**Privacy Policy** This Chrome extension, "Dark Mode Toggle", does not collect, store, or transmit any personal data from users. ### What information do we collect? None. The extension does not collect or transmit any personal information, browsing history, or user input. ### What permissions does the extension use? The extension requests the following Chrome permissions: - `activeTab` and `scripting`: to apply dark mode styles to the active website. - `storage`: to locally remember whether dark mode is enabled. - `host_permissions` (``): to allow applying styles across all websites. These permissions are used only to enable the dark mode functionality and do not result in data collection or sharing. ### Do we share data with anyone? No. This extension does not collect or share any data with third parties. ### Changes to this policy If we make changes in the future (such as to support optional cloud features), we will update this privacy policy and notify users appropriately. If you have any questions, feel free to contact us via the Chrome Web Store support page. _Last updated: [21-14-2025]_
